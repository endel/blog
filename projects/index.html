Mazmorra.io: Retro Multiplayer Roguelite RPG

Mazmorra.io has been written in pure JavaScript for the client-side - using THREE.js to render the graphics, and TypeScript for the server-side. Each graphical asset of the game is a named layer in a PSD file, which is exported via a Ruby script, and then turned into a spritesheet during the compilation process.

This project has a long history with Colyseus. It has started to validate very early vesions of Colyseus, as you can see in commits dating back from 2015, when the version of the framework was 0.2.

The source-code of Mazmorra.io was made available since July 2020.

Colyseus: Multiplayer Game Server for Node.js

The very first version of Colyseus was open-sourced back in 2015. Most of its core principles remain the same until today - which is to support mulitple game rooms ins a single server, allow developers to focus on the game logic, and be able to deliver multiplayer games without all the complexity of handling realtime state synchronization.

Hook: Backend-as-a-Service

"Hook" was a backend-as-a-service framework developed for a digital agency (Doubleleft/FLAGCX) in Brazil. Written in PHP, and using most of Laravel's ecosystem of packages, it allowed a small team of front-end developers to deliver high quality digital campaigns and installations with very tight deadlines.

Years active: 2013 - 2015.
